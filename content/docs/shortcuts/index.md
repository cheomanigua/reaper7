### Track Editing

- Split: `S` 
- Toggle Snap: `CTRL` + `S`
- Group Tracks: `G`
- Ungroup Tracks: `U`
- Add a marker: `M`

### Track Modification

- Dynamic split: `D`
- Normalize: `CRTL` + `SHIFT` + `N`
- Create Volume Envelope: `SHIFT` + Mouse click on a track


### Transport

- Play/stop: `SPACE`
- Go to the start of the timeline: `W` or `Home`
- Go to the end of the timeline: `End`
- Record: `CTRL` + `R`
- Toggle Repeat (Looping): `R`
